# Machine Learning Projects Collection

This repository contains my Machine Learning and Data Science projects built using Python, Scikit-learn, Pandas, NumPy, FastAPI, and Streamlit.

## Projects

### 1. Student Performance Predictor
- Predicts student final grades based on attendance, study hours, previous grades, and other factors.
- Tech Stack:
  - Python
  - Pandas
  - NumPy
  - Scikit-learn
  - Streamlit
- Model Used:
  - Linear Regression

---

### 2. Gurgaon House Price Prediction
- Predicts house prices in Gurgaon based on location, area, bedrooms, bathrooms, and other features.
- Tech Stack:
  - Python
  - FastAPI
  - SQLAlchemy
  - MySQL
  - Scikit-learn
- Features:
  - User Registration & Login
  - JWT Authentication
  - Prediction History
  - REST API
  - Database Integration

---

### 3. AI Resume Analyzer
- Analyzes resumes and provides feedback on skills, keywords, and ATS compatibility.
- Tech Stack:
  - Python
  - NLP
  - FastAPI
  - Machine Learning

# Gurgaon House Price Predictor

A Machine Learning web application that predicts house prices in Gurgaon.

## Tech Stack

- Python
- FastAPI
- Scikit-Learn
- Pandas
- NumPy
- HTML/CSS/JavaScript

## Features

- House price prediction
- FastAPI backend
- Feature engineering pipeline
- Interactive frontend

## Run Locally

```bash
pip install -r requirements.txt
uvicorn app:app --reload
